# 合同模板管理系统

在线地址：https://hufelix765-alt.github.io/my-hetong/

> **GitHub 打不开？** 请阅读 **[DEPLOY.md](./DEPLOY.md)**（完整部署指南）

## 本地运行

```powershell
npm install
npm run dev
```

http://localhost:5173

## 发布到 GitHub（3 步）

```powershell
npm run publish:github    # 1. 构建静态网页到 docs/
# 2. GitHub Desktop → Push origin（或 git push origin main）
# 3. GitHub → Settings → Pages → main / docs
```

详细步骤见 **[DEPLOY.md](./DEPLOY.md)**
